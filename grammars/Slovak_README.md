# Slovak Language Grammar

Comprehensive Slovak grammar covering West Slavic features including six cases and rhythmic law.

## Features
- 6 grammatical cases
- Aspect system (perfective/imperfective)
- Rhythmic law (vowel length alternation)
- Free word order

## Unique: Rhythmic Law
Two consecutive long syllables are avoided by shortening the second.
