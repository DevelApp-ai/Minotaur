# Polish Language Grammar

Comprehensive Polish grammar covering all major Slavic language features including the complete seven-case system.

## Grammar Size & Scope

- **320 lines** of comprehensive grammar rules
- **7 grammatical cases**: Nominative, Genitive, Dative, Accusative, Instrumental, Locative, Vocative
- **Aspect system**: Perfective and imperfective verb pairs
- **3 genders** with masculine personal distinction
- **Free word order** with SVO as default

## Unique Polish Features

### 1. Seven-Case System
Polish uses all seven Slavic cases:
- **Nominative**: Subject (kto? co?)
- **Genitive**: Possession, negation (kogo? czego?)
- **Dative**: Indirect object (komu? czemu?)
- **Accusative**: Direct object (kogo? co?)
- **Instrumental**: Means, accompaniment (kim? czym?)
- **Locative**: Location (o kim? o czym?)
- **Vocative**: Direct address (!)

### 2. Verb Aspect
Binary aspectual system:
- **Imperfective**: Ongoing, habitual actions (czytać - to read)
- **Perfective**: Completed actions (przeczytać - to read through)

### 3. Gender System
Three genders with masculine subdivision:
- **Masculine personal**: Male humans (mężczyzna, chłopiec)
- **Masculine animate**: Male animals (pies, kot)
- **Masculine inanimate**: Objects (stół, dom)
- **Feminine**: (kobieta, książka)
- **Neuter**: (dziecko, okno)

### 4. Past Tense Gender Agreement
Past tense forms agree with subject gender:
- **Masculine**: byłem, czytałem
- **Feminine**: byłam, czytałam
- **Neuter**: było, czytało

## Core Features

### Irregular Verbs
być (to be), mieć (to have), móc (can), chcieć (to want), musieć (must)

### Preposition Case Government
- **Genitive**: od, do, z, bez, dla, obok
- **Dative**: dzięki, ku, przeciw
- **Accusative**: na, przez, po, o, za, w
- **Instrumental**: z, nad, pod, przed, za, między
- **Locative**: w, na, o, po, przy
