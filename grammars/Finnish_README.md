# Finnish Language Grammar (Suomen kieli)

## Overview

This is a comprehensive Finnish language grammar specification matching the Danish grammar's level of detail. Finnish (Suomi) is a Uralic language spoken by approximately 5.5 million people, primarily in Finland where it is an official language alongside Swedish.

**Grammar Size**: ~1,400 lines  
**Language Family**: Uralic → Finnic  
**ISO Code**: fi  
**Speakers**: ~5.5 million  
**Official Status**: Finland, European Union

## Key Linguistic Features

Finnish is fundamentally different from Indo-European languages, with unique features including:

### 1. Fifteen Grammatical Cases

Unlike Germanic languages with 0-4 cases, Finnish has **15 grammatical cases**:

#### Core Cases
1. **Nominative** (perusmuoto): Subject, basic form
   - `talo` (house)
   
2. **Genitive** (-n): Possession, object of certain verbs
   - `talon` (of the house, house's)
   
3. **Partitive** (-a/-ä, -ta/-tä): Indefinite quantity, incomplete action
   - `taloa` (some house, houses)
   
4. **Accusative**: Definite total object (uses nominative or genitive)
   - `talon` (the house - as object)

#### Internal Locative Cases (inside)
5. **Inessive** (-ssa/-ssä): "in"
   - `talossa` (in the house)
   
6. **Elative** (-sta/-stä): "from inside"
   - `talosta` (from the house)
   
7. **Illative** (-an/-än, -han/-hän, -seen, etc.): "into"
   - `taloon` (into the house)

#### External Locative Cases (on/at)
8. **Adessive** (-lla/-llä): "at, on"
   - `talolla` (at the house)
   
9. **Ablative** (-lta/-ltä): "from (off of)"
   - `talolta` (from the house)
   
10. **Allative** (-lle): "onto, to"
    - `talolle` (to the house)

#### Other Cases
11. **Essive** (-na/-nä): "as, in the capacity of"
    - `talona` (as a house)
    
12. **Translative** (-ksi): "into (becoming)"
    - `taloksi` (into a house)
    
13. **Abessive** (-tta/-ttä): "without"
    - `talotta` (without a house)
    
14. **Instructive** (-in): "by means of" (plural only)
    - `taloin` (by means of houses)
    
15. **Comitative** (-ne- + possessive): "with" (plural only)
    - `taloineen` (with his/her houses)

### 2. Vowel Harmony

Finnish has strict vowel harmony rules where vowels in a word must be either:

**Back vowels**: a, o, u  
**Front vowels**: ä, ö, y  
**Neutral vowels**: e, i (can appear with either)

Examples:
- `talossa` (in the house) - back harmony
- `kädes` (in the hand) - front harmony
- `tiedossa` (in knowledge) - neutral + back

All suffixes adapt to the stem's vowel harmony:
- Partitive: `-a` (back) vs. `-ä` (front)
- Inessive: `-ssa` (back) vs. `-ssä` (front)

### 3. Agglutinative Morphology

Finnish builds complex meanings by adding multiple suffixes:

```
talo + i + ssa + ni + ko
house + plural + inessive + my + question
"in my houses?"
```

Example breakdown:
- `taloissanikin` = talo + i + ssa + ni + kin
  - "in my houses too"

### 4. No Grammatical Gender

Unlike Germanic languages, Finnish has:
- No grammatical gender (no masculine/feminine/neuter)
- Same pronoun `hän` for "he" and "she"
- Only semantic gender distinctions

### 5. SOV Word Order Tendency

Basic word order: Subject-Object-Verb (flexible due to cases)

```
Minä juon kahvia.
I drink coffee.
(But word order can vary: "Kahvia minä juon" emphasizes coffee)
```

### 6. Five Infinitives

Finnish has five different infinitive forms, each with different uses:

1. **First infinitive** (-a/-ä, -da/-dä, -ta/-tä, etc.): Basic form
   - `puhua` (to speak)
   
2. **Second infinitive** (-essa/-essä, -en): Temporal/instrumental
   - `puhuessa` (while speaking)
   
3. **Third infinitive** (-ma/-mä + cases): Various functions
   - `puhumassa` (in the act of speaking)
   - `puhumaan` (in order to speak)
   
4. **Fourth infinitive** (-minen): Verbal noun
   - `puhuminen` (speaking, the act of speaking)
   
5. **Fifth infinitive** (-maisa/-mäisä): "about to"
   - `puhemaisa` (about to speak - rare)

### 7. Passive Voice

Finnish passive is impersonal (no explicit subject):

```
Täällä puhutaan suomea.
Here is-spoken Finnish.
"Finnish is spoken here" / "One speaks Finnish here"
```

### 8. Extensive Participle System

Four main participles:
- **Present active**: `-va/-vä` (speaking)
- **Present passive**: `-tava/-tävä` (being spoken)
- **Past active**: `-nut/-nyt` (spoken/having spoken)
- **Past passive**: `-ttu/-tty` (been spoken)
- **Agent participle**: `-ma/-mä` (one who has done)
- **Negative participle**: `-maton/-mätön` (not done)

## Regional Dialects

Finnish has significant dialectal variation across six major dialect groups:

### 1. Western Finnish Dialects (Länsisuomalaiset murteet)

**Region**: Southwestern and western Finland  
**Pronouns**: `mää, sää, myö, työ` (minä, sinä, me, te)  
**Features**:
- 3pl verb ending: `-aan/-ään` instead of `-vat/-vät`
- Dropping of final `-n` in some words

**Example**:
```
Standard: "He puhuvat suomea"
Western: "He puhvaan suomee"
(They speak Finnish)
```

### 2. Eastern Finnish Dialects (Itäsuomalaiset murteet)

**Region**: Eastern and northeastern Finland  
**Pronouns**: `mie, sie, myö, työ`  
**Features**:
- Preservation of diphthongs: `ie`, `uo`, `yö`
- Palatalization
- Vowel changes in diphthongs

**Example**:
```
Standard: "Minä menen kotiin"
Eastern: "Mie menen kottii"
(I go home)
```

### 3. Northern Finnish Dialects (Pohjoissuomalaiset murteet)

**Region**: Northern Finland, Lapland  
**Pronouns**: `mää, sää`  
**Features**:
- Different 1pl ending: `-mma/-mmä` instead of `-mme`
- Simplified consonant gradation

**Example**:
```
Standard: "Me puhumme"
Northern: "Myö puhumma"
(We speak)
```

### 4. Southwestern Dialects (Lounaismurteet)

**Region**: Southwest Finland  
**Features**:
- 3sg verb ending `-a/-ä` instead of null ending
- Unique vocabulary
- Swedish influences

**Example**:
```
Standard: "Hän puhuu"
Southwest: "Hän puhuu" (similar but with different pronunciation)
```

### 5. Southeastern Dialects (Kaakkoismurteet)

**Region**: Southeastern Finland  
**Features**:
- Long vowels in certain positions: `oo`, `ee`
- Distinctive intonation

### 6. Middle Finnish Dialects (Keskisuomalaiset murteet)

**Region**: Central Finland  
**Features**:
- Mixed features from eastern and western dialects
- 3pl ending variations: `-haan/-hään`

## Colloquial Finnish (Puhekieli)

Spoken Finnish differs significantly from written standard:

### Colloquial Pronouns
- `mä, sä` (minä, sinä)
- `mun, sun` (minun, sinun - genitive)
- `mua, sua` (minua, sinua - partitive)
- `mut, sut` (minut, sinut - accusative)
- `mulla, sulla` (minulla, sinulla - adessive)

### Colloquial Verbs
- 3pl: `-vat/-vät` → `-aan/-ään`
  - `he puhuvat` → `ne puhuu` or `ne puhuu`
- Negative: `ei ole` → `ei oo`, `ei mene` → `ei mee`

### Colloquial Demonstratives
- `tämä` → `tää`
- `tuo` → `toi`
- `nämä` → `nää`
- `nuo` → `noi`

**Example Comparison**:
```
Standard: "Minä en tiedä, missä hän on"
Colloquial: "Mä en tiiä, mis se on"
(I don't know where he/she is)
```

## Complex Sentence Structures

### 1. Nested Subordinate Clauses

Finnish supports multiple levels of embedding:

```
Tiedän, että Maria uskoo, että hänen veljensä, joka asuu Lontoossa, tietää, missä heidän äitinsä on.
(I know that Maria believes that her brother, who lives in London, knows where their mother is.)
```

### 2. Relative Clauses

Using `joka` (who/which) and `mikä` (which/what):

```
Mies, joka asuu naapurissa, on lääkäri.
(The man who lives next door is a doctor.)
```

### 3. Conditional Clauses

With `jos` (if), `jollei` (if not), `ellei` (unless):

```
Jos sataa, emme mene ulos.
(If it rains, we won't go out.)
```

### 4. Temporal Clauses

Various temporal conjunctions:
- `kun` (when)
- `kunnes` (until)
- `ennen kuin` (before)
- `jälkeen kun` (after)

```
Kun tulen kotiin, syön illallisen.
(When I come home, I eat dinner.)
```

### 5. Cleft Sentences

For emphasis:

```
Se on Pekka, joka tuli eilen.
(It is Pekka who came yesterday.)
```

## Verb System

### Present Tense

Personal endings:
- 1sg: `-n` (puhun - I speak)
- 2sg: `-t` (puhut - you speak)
- 3sg: `Ø` or `-V` (puhuu - he/she speaks)
- 1pl: `-mme` (puhumme - we speak)
- 2pl: `-tte` (puhutte - you speak)
- 3pl: `-vat/-vät` (puhuvat - they speak)

### Past Tense

Past marker `-i-` + personal endings:
- `puhuin` (I spoke)
- `puhuit` (you spoke)
- `puhui` (he/she spoke)
- `puhuimme` (we spoke)
- `puhuitte` (you spoke)
- `puhuivat` (they spoke)

### Conditional Mood

Marker `-isi-` + personal endings:
- `puhuisin` (I would speak)
- `puhuisit` (you would speak)
- `puhuisi` (he/she would speak)
- etc.

### Potential Mood

Marker `-ne-` + personal endings (expresses possibility):
- `puhunen` (I might speak)
- `puhunet` (you might speak)
- `puhunee` (he/she might speak)

### Imperative Mood

Commands:
- 2sg: stem (puhु - speak!)
- 2pl: `-kaa/-kää` (puhukaa - speak!)
- 3sg: `-koon/-köön` (puhukoon - let him/her speak)
- 1pl: `-kaamme/-käämme` (puhukaamme - let's speak)

### Negative Forms

Negation uses auxiliary `ei` + connegative form:
- Present: `en puhu` (I don't speak)
- Past: `en puhunut` (I didn't speak)
- Conditional: `en puhuisi` (I wouldn't speak)

## Common Irregular Verbs

While Finnish is mostly regular, some verbs have stem changes:

1. **olla** (to be): `olen, olet, on, olemme, olette, ovat`
2. **tulla** (to come): `tulen, tulet, tulee, tulemme, tulette, tulevat`
3. **mennä** (to go): `menen, menet, menee, menemme, menette, menevät`
4. **tehdä** (to do/make): `teen, teet, tekee, teemme, teette, tekevät`
5. **nähdä** (to see): `näen, näet, näkee, näemme, näette, näkevät`
6. **juoda** (to drink): `juon, juot, juo, juomme, juotte, juovat`
7. **syödä** (to eat): `syön, syöt, syö, syömme, syötte, syövät`

## Partitive vs. Accusative Objects

A unique Finnish feature is the choice between partitive and accusative for objects:

### Use Partitive When:
1. Object is indefinite: `Juon kahvia` (I drink coffee - some coffee)
2. Action is incomplete: `Luin kirjaa` (I was reading a book)
3. Sentence is negative: `En juo kahvia` (I don't drink coffee)
4. With partitive verbs: `Rakastan sinua` (I love you)

### Use Accusative/Genitive When:
1. Object is definite and total: `Join kahvin` (I drank the coffee - all of it)
2. Action is complete: `Luin kirjan` (I read the book - finished)

This is one of the most challenging aspects for learners!

## Example Sentences

```

```


### Complex Sentences
```

```

**Colloquial**:
```

```

## Context-Sensitive Validation

The grammar includes 15 context-sensitive validation rules:

1. **Vowel harmony**: Back/front vowels cannot mix in suffixes
2. **Adjective-noun case agreement**: Adjectives must match noun case
3. **Subject-verb number agreement**: Subject and verb must agree
4. **Personal pronoun case selection**: Correct case for function
5. **Partitive object usage**: Rules for when to use partitive
6. **Negation agreement**: Negation requires connegative form
7. **Question particle placement**: -ko/-kö attaches correctly
8. **Relative pronoun agreement**: Number agreement with antecedent
9. **Conditional consistency**: Appropriate verb forms in conditionals
10. **Infinitive selection**: Correct infinitive with modal verbs
11. **Passive construction**: Passive is impersonal
12. **Dialect consistency**: Don't mix dialects
13. **Register consistency**: Don't mix standard and colloquial
14. **Postposition government**: Postpositions require genitive
15. **Nested clause depth**: Warnings for deep nesting
