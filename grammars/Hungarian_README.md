# Hungarian Language Grammar

Comprehensive Hungarian grammar covering unique Uralic agglutinative features including 18+ case suffixes and vowel harmony.

## Grammar Size & Scope

- **380+ lines** of comprehensive grammar rules
- **18+ grammatical cases**: All expressed through agglutinative suffixes
- **No grammatical gender**: Simplifies agreement rules
- **Vowel harmony**: Front vs back vowels determine suffix forms
- **Definite vs indefinite conjugation**: Verbs change based on object definiteness
- **Agglutinative morphology**: Extensive suffix stacking

## Unique Hungarian Features

### 1. Agglutinative Case System (18+ Cases)
Cases are expressed through suffixes stacked on noun stems:
- **Nominative**: könyv (book)
- **Accusative**: könyvet (book-ACC)
- **Dative**: könyvnek (to-book)
- **Instrumental**: könyvvel (with-book)
- **Inessive**: könyvben (in-book)
- **Superessive**: könyvön (on-book)
- **Adessive**: könyvnél (at-book)
- **Illative**: könyvbe (into-book)
- **Sublative**: könyvre (onto-book)
- **Allative**: könyvhöz (to/towards-book)
- **Elative**: könyvből (out-of-book)
- **Delative**: könyvről (from-on-book)
- **Ablative**: könyvtől (away-from-book)
- Plus: Causal-final, Translative, Terminative, Essive-formal, Essive-modal

### 2. Vowel Harmony
Suffixes change based on whether the stem contains back or front vowels:
- **Back vowels**: a, á, o, ó, u, ú → back suffix variants (-ban, -nak, -val)
- **Front vowels**: e, é, i, í, ö, ő, ü, ű → front suffix variants (-ben, -nek, -vel)
- Example: házban (in house) vs kertben (in garden)

### 3. Definite vs Indefinite Conjugation
Verbs have two conjugation paradigms:
- **Indefinite**: Used with indefinite objects or no object
  - látok (I see), látsz (you see)
- **Definite**: Used with definite objects
  - látom (I see it/the X), látod (you see it/the X)

### 4. No Grammatical Gender
Unlike most European languages, Hungarian has no grammatical gender:
- Same pronoun "ő" for he/she/it
- No gender agreement needed

### 5. Postpositions (Not Prepositions)
Hungarian uses postpositions that follow the noun:
- **a ház mellett** (next-to the house) not *next to the house*
- **számomra** (for me)

### 6. SOV Word Order (Topic-Prominent)
Default word order is Subject-Object-Verb:
- **Én a könyvet olvasom** (I the book read)
- But highly flexible for topic/focus marking

## Core Features

### Irregular Verbs
lenni/van (to be), menni (to go), jönni (to come), tenni (to put), venni (to buy)

### Modal Verbs
akar (want), tud (can/know), kell (must), szabad (may), lehet (can/possible)

### Possessive Suffixes
Can be stacked with case suffixes:
- könyvem (my book)
- könyvemben (in my book)
- könyveimben (in my books)
