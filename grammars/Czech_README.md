# Czech Language Grammar

Comprehensive Czech grammar covering all major Slavic language features including the complete seven-case system and vowel length distinctions.

## Grammar Size & Scope

- **350+ lines** of comprehensive grammar rules
- **7 grammatical cases**: Nominative, Genitive, Dative, Accusative, Instrumental, Locative, Vocative
- **Aspect system**: Perfective and imperfective verb pairs
- **3 genders** with masculine animate/inanimate distinction
- **Vowel length distinctions**: Short vs long vowels (á/a, é/e, í/i, ó/o, ú-ů/u, ý/y)

## Unique Czech Features

### 1. Masculine Animate vs Inanimate Distinction
Czech distinguishes between animate and inanimate masculine nouns:
- **Animate accusative = genitive**: Vidím studenta (I see the student)
- **Inanimate accusative = nominative**: Vidím stůl (I see the table)

### 2. Vowel Length Phonemic Opposition
Vowel length is phonemically distinctive:
- **krátký** (short) vs **krátky** (different meaning based on context)
- **d** vs **dá** (gives)
- **byl** (was-masculine) vs **být** (to be)

### 3. Complex Consonant Clusters
Czech allows extensive consonant clusters:
- **scvrkl** (shriveled)
- **zmrzl** (frozen)
- **strč prst skrz krk** (stick finger through neck - tongue twister)

### 4. Free Word Order with SVO Default
While SVO is default, word order is highly flexible for emphasis:
- **Student čte knihu** (The student reads a book)
- **Knihu čte student** (It's a book that the student reads)
- **Čte student knihu** (The student does read a book)

## Core Features

### Seven-Case System
- **Nominative**: Subject (kdo? co?)
- **Genitive**: Possession, negation (koho? čeho?)
- **Dative**: Indirect object (komu? čemu?)
- **Accusative**: Direct object (koho? co?)
- **Instrumental**: Means, accompaniment (kým? čím?)
- **Locative**: Location (o kom? o čem?)
- **Vocative**: Direct address

### Irregular Verbs
být (to be), mít (to have), moci (can), chtít (to want), vědět (to know), dát (to give), vzít (to take)

### Preposition Case Government
- **Genitive**: od, do, z, bez, pro, vedle, u
- **Dative**: k, proti, díky, kvůli
- **Accusative**: na, přes, po, o, za, v
- **Instrumental**: s, nad, pod, před, za, mezi
- **Locative**: v, na, o, po, při

### Past Tense Gender Agreement
Past tense forms agree with subject gender:
- **Masculine**: byl, četl
- **Feminine**: byla, četla
- **Neuter**: bylo, četlo
