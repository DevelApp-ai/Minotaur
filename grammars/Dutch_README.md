# Dutch Language Grammar

Comprehensive Dutch grammar covering all major language features including unique Dutch characteristics.

## Unique Dutch Features

### 1. V2 Word Order (Verb-Second)
Main clauses have the finite verb in second position:
- Ik **werk** vandaag (I work today)
- Vandaag **werk** ik (Today work I)

### 2. SOV in Subordinate Clauses
Subordinate clauses have verb-final order:
- omdat ik vandaag **werk** (because I today work)
- als hij morgen **komt** (if he tomorrow comes)

### 3. Separable Prefix Verbs
Verbs with separable prefixes split in main clauses:
- opbellen → Ik **bel** je **op** (I call you up)
- meekomen → Hij **komt** **mee** (He comes along)

### 4. Diminutives
Unique productive diminutive system (always neuter):
- huis → huisje (little house)
- boek → boekje (little book)
- Kind → kindje (little child)

### 5. Two Genders (Common + Neuter)
- **Common (de)**: de man, de vrouw
- **Neuter (het)**: het kind, het huis
- **Plural (de)**: de mannen, de kinderen

## Core Features

### Verb System
- **Present**: werk, werkt, werken
- **Past**: werkte, werkten (weak) / kwam, kwamen (strong)
- **Perfect**: heb gewerkt, ben gekomen
- **Future**: zal werken

### Major Irregular Verbs
zijn, hebben, kunnen, mogen, moeten, willen, zullen, gaan, komen, doen, zien, geven

### Adjective Inflection
- **Uninflected**: een groot huis (neuter, indefinite)
- **Inflected**: de grote man (common, definite)
- **Inflected**: grote huizen (plural)
